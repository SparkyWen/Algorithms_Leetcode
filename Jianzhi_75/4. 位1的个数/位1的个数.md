# 位1的个数

## 1. 算法解释

> 1. bin 方法
>
> ```python
> bin() 函数可以得到一个整数的二进制字符串。比如 bin(666) 会得到：
> 
> >>> bin(666)
> '0b1010011010'
> ```
>
> 2. 进位法
>
> ```python
> class Solution:
>     def hammingWeight(self, n: int) -> int:
>         res = 0
>         while n:
>             res += n & 1
>             n >>= 1
>         return res
> 
> ```
>
> ```python
> 一、每行在做什么？
> 
> res = 0
> 用于累计 “1 的个数”。
> 
> while n:
> 当 n 不为 0 时循环。每次循环处理 n 的最低位，然后把 n 右移一位，直到所有位都被处理完（n 变成 0）。
> 
> res += n & 1
> & 是按位与。n & 1 只会取出 n 的最低位：
> 
> 如果最低位是 1，则 n & 1 == 1；
> 
> 如果最低位是 0，则 n & 1 == 0。
> 把它加到 res 上，相当于：最低位是 1 就计数 +1，否则 +0。
> 
> n >>= 1
> 把 n 右移一位，丢掉当前最低位，让“次低位”成为新的最低位，下一轮再检测。
> 在 Python 里，对非负数 >> 是算术右移与逻辑右移等价（高位补 0）；（只有负数时会在高位补 1，后面有注意点）。
> 
> return res
> 循环结束（n 变为 0），返回累计到的 1 的个数。
> 
> 
> ```

> 3. 解释过程：

| 轮次 | n（二进制） | n & 1 | res 累计 | 右移后 n |
| ---- | ----------- | ----- | -------- | -------- |
| 1    | 1011 (11)   | 1     | 1        | 0101 (5) |
| 2    | 0101 (5)    | 1     | 2        | 0010 (2) |
| 3    | 0010 (2)    | 0     | 2        | 0001 (1) |
| 4    | 0001 (1)    | 1     | 3        | 0000 (0) |

## 2. 代码

class Solution(object):

```python
class Solution(object):
    def hammingWeight(self, n):
        res = 0
        while n:
            res += n & 1
            n >>= 1
        return res
```
